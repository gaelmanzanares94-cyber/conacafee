Más información

Resumen: este paquete ha sido preparado para facilitar su conversión a APK. Se eliminaron archivos .svg que pueden causar problemas en algunos convertidores automáticos.

Cómo usar

  1. Revisar la carpeta assets/ y añadir imágenes PNG/JPG en lugar de SVG si las necesita.
  2. Asegúrese de que index.html esté en la raíz del paquete antes de generar el APK.
  3. Si usa herramientas como Capacitor o Cordova, corra la compilación en una máquina con Node.js y Android SDK.
