
CONALEP - Sistema de pedidos (Entrega final)
Incluye: productos con imágenes y ofertas, login por matrícula, carrito, pago simulado (QR/tarjeta), generación de código de recogida y descarga de ticket en PDF, PWA básica.

Instrucciones:
1. Descomprime la carpeta.
2. Abrir index.html en un navegador moderno. Se recomienda Live Server en VS Code.
3. Entrar con nombre y matrícula, seleccionar productos, pagar y descargar ticket.


Nota: archivo revisado y actualizado para compatibilidad con conversión a APK. Se recomienda revisar imágenes en assets/.
